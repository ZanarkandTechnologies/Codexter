# Hardcases

Sanitized `repent hardcase` captures live here.

Each case should be created under:

```text
experiments/hardcases/YYYYMMDD-HHMM-<slug>/case.md
```

Keep raw private transcripts, secrets, credentials, system/developer prompts,
encrypted reasoning, and unrelated personal data out of this directory. Store
short evidence excerpts and local artifact pointers instead.
