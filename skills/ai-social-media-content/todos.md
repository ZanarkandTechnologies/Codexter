# AI Social Media Content Todos

Use this as the ordered checklist whenever `ai-social-media-content` is active.

- [ ] Classify the requested artifact: TikTok/Reels/Shorts video, feed image, carousel, thumbnail, caption set, hashtag set, UGC concept, content calendar, or cross-platform campaign bundle.
- [ ] State the target platform set, audience, format, output count, dimensions, tone, and handoff path before producing content.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when examples, platform specs, brand/source assets, current model behavior, platform norms, peer posts, or campaign expectations should guide scope.
- [ ] Use [plan](../plan/SKILL.md) when choosing platforms, formats, hooks, visual routes, asset routes, or scope cuts.
- [ ] Load the shared [image/social production workflow](../image-generation/references/domain-production.md).
- [ ] Route carousel work through [social-media-carousel](../social-media-carousel/SKILL.md).
- [ ] Route still visuals through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route video clips through [video-generation](../video-generation/SKILL.md), or through [remotion](../remotion/SKILL.md) and [remotion-render](../remotion-render/SKILL.md) for deterministic code-rendered video.
- [ ] Route campaign bundle or web integration through [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Draft copy, captions, prompts, or asset plans before generation or rendering.
- [ ] Save drafts, prompts, inputs, result JSON, generated files, and notes inside the workspace when external generation is involved.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, post, schedule, or cross-post unless the user explicitly asks for that action.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming platform fit, campaign readiness, or final quality.
