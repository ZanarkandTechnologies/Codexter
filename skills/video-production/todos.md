# Video Production Todos

Use this as the ordered checklist whenever `video-production` or one of its
method addresses is active.

- [ ] Select one primary method: `video-production:marketing`,
  `video-production:explainer`, `video-production:storyboard`,
  `video-production:talking-head`, or `video-production:ad-spec`. Add a
  supporting method only when the deliverable genuinely spans formats.
- [ ] Classify the video job: launch, promo, product demo, explainer,
  tutorial, storyboard, shot list, animatic, talking-head, avatar/lipsync,
  testimonial, platform ad, UGC ad, video edit, or campaign bundle.
- [ ] State the audience, channel/platform, duration, aspect ratio, CTA,
  source assets, likeness/consent boundaries, output files, asset needs, and
  handoff path before producing scripts or assets.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or
  [research:parity](../research/SKILL.md#researchparity) when examples,
  platform specs, source assets, current model behavior, peer videos,
  category norms, creative expectations, or production standards should guide
  scope.
- [ ] Use [plan](../plan/SKILL.md) when choosing campaign angle, structure,
  visual carrier, model family, platform cut, production route, or scope
  boundary.
- [ ] Load the shared [domain video production workflow](../video-generation/references/domain-production.md).
- [ ] For `video-production:marketing`, load
  [marketing upstream](references/upstream-marketing.md) or
  [marketing prompting](references/prompting-marketing.md) when campaign
  frameworks, shot types, templates, launch/product/testimonial prompts, or
  social promo patterns matter.
- [ ] For `video-production:explainer`, load
  [explainer upstream](references/upstream-explainer.md) or
  [explainer prompting](references/prompting-explainer.md) when script
  formulas, pacing, voiceover, UI walkthroughs, feature demos, or motion
  graphics prompts matter.
- [ ] For `video-production:storyboard`, load
  [storyboard upstream](references/upstream-storyboard.md) or
  [storyboard prompting](references/prompting-storyboard.md) when shot
  vocabulary, panel sequence, continuity, visual scripts, or image-to-video
  motion prompts matter.
- [ ] For `video-production:talking-head`, load
  [talking-head upstream](references/upstream-talking-head.md) or
  [talking-head prompting](references/prompting-talking-head.md) when portrait
  requirements, scripts, avatar behavior, voice prompts, camera/background, or
  lipsync prompts matter.
- [ ] For `video-production:ad-spec`, load
  [ad-spec upstream](references/upstream-ad-spec.md) or
  [ad-spec prompting](references/prompting-ad-spec.md) when platform specs,
  safe zones, duration limits, hooks, captions, UGC prompts, or CTA prompts
  matter.
- [ ] Draft scripts, shot lists, panel sequences, ad specs, caption/safe-zone
  notes, prompt sets, asset lists, and proof plans before generation or
  rendering.
- [ ] Route model-native clips through [video-generation](../video-generation/SKILL.md).
- [ ] Route still assets, posters, product frames, portraits, or references
  through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route deterministic motion graphics or code-rendered assembly through
  [remotion](../remotion/SKILL.md) and
  [remotion-render](../remotion-render/SKILL.md).
- [ ] Route website, landing-page, or campaign-bundle integration through
  [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Save briefs, scripts, storyboards, ad specs, prompts, inputs, result
  JSON, generated files, source-asset notes, and review notes inside the
  workspace when external generation is involved.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly
  acceptable before running model or `belt` jobs.
- [ ] Do not publish, post, upload, buy media, spend media budget, or represent
  generated identity assets as approved unless the user explicitly authorizes
  that use.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before
  claiming narrative quality, continuity, platform fit, likeness/voice quality,
  creative quality, render quality, or final production quality.
