# Workflows

## Visual Direction

1. Confirm the functional basis.
2. Classify product vs brand register.
3. Write the scene sentence.
4. Set taste dials.
5. Define typography, color, spacing, radius/elevation, media, and motion vocabulary.
6. Run anti-slop checks.
7. Hand implementation constraints to `frontend-craft`.

## Visual Polish

1. Diagnose hierarchy, rhythm, typography, color, responsiveness, and AI tells.
2. Preserve existing design-system constraints.
3. Recommend the smallest visual change that fixes the main issue.
4. Hand off concrete implementation constraints.
