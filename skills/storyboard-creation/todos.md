# Storyboard Creation Todos

Use this as the ordered checklist whenever `storyboard-creation` is active.

- [ ] Classify the board: shot list, visual script, panel board, animatic plan, ad storyboard, product storyboard, or image-to-video motion test.
- [ ] State the story goal, audience, format, panel count, aspect ratio, continuity needs, output files, and handoff path before making panels.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when examples, shot vocabulary, source assets, continuity rules, current model behavior, peer storyboards, ad formats, or production standards should guide the board.
- [ ] Use [plan](../plan/SKILL.md) when choosing storyboard format, shot language, visual route, panel granularity, or scope cut.
- [ ] Load the shared [domain video production workflow](../video-generation/references/domain-production.md).
- [ ] Load [storyboard prompting](references/prompting.md) before writing panel prompts, shot lists, visual continuity prompts, or image-to-video motion prompts.
- [ ] Draft the beat list, shot vocabulary, panel sequence, and continuity checks before generating final panels.
- [ ] Route panel stills through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route motion tests or clips through [video-generation](../video-generation/SKILL.md), or through [remotion](../remotion/SKILL.md) and [remotion-render](../remotion-render/SKILL.md) for deterministic animatics.
- [ ] Route web or campaign asset integration through [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Save panel prompts, inputs, result JSON, generated files, shot notes, and continuity notes inside the workspace.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, post, or upload generated storyboard assets unless the user explicitly asks for that action.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming continuity, panel quality, or production readiness.
