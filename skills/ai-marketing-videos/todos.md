# AI Marketing Videos Todos

Use this as the ordered checklist whenever `ai-marketing-videos` is active.

- [ ] Classify the campaign job: launch, feature highlight, testimonial, before/after, brand story, UGC ad, product demo, or paid creative.
- [ ] State the target audience, channel, duration, aspect ratio, CTA, and final handoff format before planning shots.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when examples, platform specs, source assets, current model behavior, campaign norms, peer ads, or platform expectations should guide the brief.
- [ ] Use [plan](../plan/SKILL.md) when choosing among campaign angles, visual carriers, model families, scope cuts, or output routes.
- [ ] Load the shared [domain video production workflow](../video-generation/references/domain-production.md).
- [ ] Load [marketing video prompting](references/prompting.md) before writing launch, product, testimonial, brand, or social promo prompts.
- [ ] Route model-native clips through [video-generation](../video-generation/SKILL.md).
- [ ] Route still assets, posters, product frames, or references through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route deterministic motion graphics or code-rendered assembly through [remotion](../remotion/SKILL.md) and [remotion-render](../remotion-render/SKILL.md).
- [ ] Route website, landing-page, or campaign-bundle integration through [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Save the brief, prompts, inputs, result JSON, generated files, and notes inside the workspace when external generation is involved.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, post, buy media, or launch campaigns unless the user explicitly asks for that action.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming final creative, platform-fit, or production quality.
