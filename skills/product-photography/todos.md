# Product Photography Todos

Use this as the ordered checklist whenever `product-photography` is active.

- [ ] Classify the shot set: hero, packshot, lifestyle, scale, detail, in-use, variation, cutout, background removal, upscale, mockup, or ad image.
- [ ] State the product facts, source assets, marketplace/channel, shot count, aspect ratios, style, and final handoff path before generation.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when product examples, marketplace specs, source assets, current model behavior, e-commerce norms, marketplace norms, or peer product-photo norms should guide the brief.
- [ ] Use [plan](../plan/SKILL.md) when choosing shot sets, backgrounds, model families, realism tradeoffs, or output routes.
- [ ] Load the shared [image/social production workflow](../image-generation/references/domain-production.md).
- [ ] Load [upstream product photography](references/upstream-product-photography.md) when shot types, marketplace constraints, or product-image conventions matter.
- [ ] Load [upstream AI product photography](references/upstream-ai-product-photography.md) when generated product-photo model patterns matter.
- [ ] Use `imagegen` for normal Codex-native bitmap generation or editing.
- [ ] Use [image-generation](../image-generation/SKILL.md) for named inference.sh image models, background removal, upscaling, CLI repeatability, or structured result bundles.
- [ ] Route product-page or frontend integration through [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Save source assets, prompts, inputs, result JSON, final images, and notes inside the workspace when external generation is involved.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, upload, or change store listings unless the user explicitly asks for that action.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming commercial, marketplace, or product-clarity quality.
