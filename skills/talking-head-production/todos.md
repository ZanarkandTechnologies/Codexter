# Talking Head Production Todos

Use this as the ordered checklist whenever `talking-head-production` is active.

- [ ] Classify the talking-head type: avatar presenter, lipsync, AI spokesperson, course segment, product demo, testimonial, social presenter clip, or portrait animation.
- [ ] State the presenter/source asset, script, voice/audio need, background, duration, aspect ratio, output format, and handoff path before generation.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when examples, portrait requirements, source assets, model behavior, platform specs, peer presenters, avatar norms, or production expectations should guide scope.
- [ ] Use [plan](../plan/SKILL.md) when choosing presenter type, avatar/model route, script style, visual carrier, or scope cut.
- [ ] Load the shared [domain video production workflow](../video-generation/references/domain-production.md).
- [ ] Load [talking-head prompting](references/prompting.md) before writing avatar behavior prompts, voice prompts, portrait prompts, camera/background instructions, or lipsync scripts.
- [ ] Keep consent, likeness, brand, and source-asset boundaries explicit when a real person or customer asset is involved.
- [ ] Route still portraits or references through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route avatar, lipsync, portrait animation, or model-native presenter clips through [video-generation](../video-generation/SKILL.md).
- [ ] Save scripts, prompts, inputs, source asset notes, result JSON, generated files, and review notes inside the workspace.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, post, upload, or represent generated identity assets as approved unless the user explicitly authorizes that use.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming likeness, voice, platform-fit, or final production quality.
