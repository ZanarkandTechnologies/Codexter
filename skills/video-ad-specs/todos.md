# Video Ad Specs Todos

Use this as the ordered checklist whenever `video-ad-specs` is active.

- [ ] Classify the ad surface: TikTok, Instagram Reels/Stories, YouTube Shorts, bumper, pre-roll, Facebook, LinkedIn, UGC ad, product ad, or mobile paid social.
- [ ] State platform, placement, dimensions, duration, safe zones, caption needs, hook, CTA, deliverables, and handoff path before producing creative.
- [ ] Use [research:competitor](../research/SKILL.md#researchcompetitor) or [research:parity](../research/SKILL.md#researchparity) when platform specs, peer ads, source assets, current model behavior, ad category norms, competitor creative, or platform expectations should guide the brief.
- [ ] Use [plan](../plan/SKILL.md) when choosing hook, framework, visual route, model family, platform cut, or scope boundary.
- [ ] Load the shared [domain video production workflow](../video-generation/references/domain-production.md).
- [ ] Load [video ad prompting](references/prompting.md) before writing platform-specific hooks, visual scenes, UGC prompts, caption-safe shots, or CTA prompts.
- [ ] Draft the ad spec, shot list, caption/safe-zone notes, asset list, and proof plan before generation.
- [ ] Route broader marketing video work through [ai-marketing-videos](../ai-marketing-videos/SKILL.md).
- [ ] Route model-native clips through [video-generation](../video-generation/SKILL.md).
- [ ] Route still assets through `imagegen` or [image-generation](../image-generation/SKILL.md).
- [ ] Route campaign web integration through [frontend-craft](../frontend-craft/SKILL.md).
- [ ] Save ad specs, prompts, inputs, result JSON, generated files, and notes inside the workspace when external generation is involved.
- [ ] Confirm external compute, spend, uploads, or API usage is explicitly acceptable before running model or `belt` jobs.
- [ ] Do not publish, upload, run ads, or spend media budget unless the user explicitly asks for that action.
- [ ] Follow the [execute](../execute/SKILL.md) proof and writeback loop before claiming platform-fit, safe-zone, creative, or final ad quality.
