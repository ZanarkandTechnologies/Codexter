# Workflows

## Standard Landing Page

1. Define offer and audience.
2. Choose story arc.
3. Map sections.
4. Plan visual scenes and assets.
5. Set visual rules through `visual-design`.
6. Plan motion only where it strengthens the story.
7. Define QA checkpoints.
8. Hand off to `frontend-craft`.

## Cinematic Scrolltelling

1. Define narrative phases.
2. Choose pinned/sticky or native-scroll structure.
3. Plan media layers.
4. Route complex timelines to official GreenSock skills or docs.
5. Verify desktop/mobile/reduced-motion checkpoints.
