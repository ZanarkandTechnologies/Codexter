# Video Generation Skill Instructions

This module is a category-level skill. Keep the first-load `SKILL.md` focused on routing, capability gates, output handling, and proof.

The files under `references/tools/` and `references/guides/` are copied upstream inference.sh `SKILL.md` files. Refresh them from the upstream source instead of rewriting them by hand.

Use `todos.md` as the anti-forgetting checklist for selecting references.

Keep `remotion-render` separate. Model-native generation and code-to-video rendering have different inputs, failure modes, and QA expectations.
