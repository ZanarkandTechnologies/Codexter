# `skills/deep-interview/AGENTS.md`

Rules for the `deep-interview` skill surface.

## Purpose

`skills/deep-interview/` owns the ambiguity-reduction contract that sharpens a
vague ask before planning or execution.

## Keep

- one-question-at-a-time clarification
- ticket/docs-first grounding when a live ticket already exists
- requirements summaries on current Farplane surfaces
- explicit non-goals and decision boundaries before handoff

## Do Not

- create hidden sidecar context artifacts outside the active ticket, docs, or
  mode state
- treat transcript memory as the canonical durable surface
- implement directly from inside the interview mode
