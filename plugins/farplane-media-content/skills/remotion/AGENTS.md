# Remotion Skill

Use this folder for Remotion authoring guidance imported from the official Remotion skill package.

- Keep `SKILL.md` as the public routing and first-load surface.
- Keep detailed domain rules under `rules/` and load them only when the task needs that specific Remotion feature.
- Use `remotion-render` only when the authored Remotion code needs inference.sh MP4 rendering.
