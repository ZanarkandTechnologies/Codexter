# Judgement Calls

Every synthesis has decisions that cannot be solved by counting alone. Use
`advise` for those decisions and record the answer in the decision matrix.

## Judgement Call Card

```text
Decision:
Options:
Criteria:
Recommended option:
Why it wins:
Tradeoff accepted:
Metric or evidence to revisit later:
```

## Common Questions

- Which source should set the standard?
- Which feature is overfitted to its original product?
- Which metric best reflects the user job?
- Which impressive idea should be rejected?
- Which guard prevents the primary metric from being gamed?
- What is the smallest credible slice?

## Skill-Level Pattern

When creating or updating a skill, add 3-7 judgement questions when the skill
requires material choices. The questions should tell the agent when to use
`advise` instead of pretending the answer is deterministic.
