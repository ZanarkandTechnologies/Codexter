# Architecture

`frontend-design` is now an app-UI implementation reference layer.

It owns:

- shadcn setup,
- AI Elements,
- curated registries,
- theming,
- component construction guidance.

It does not own:

- UX/workflow planning (`functional-ui`),
- visual taste and system direction (`visual-design`),
- one-page/scrolltelling planning (`landing-page`),
- end-to-end implementation orchestration (`frontend-craft`).
