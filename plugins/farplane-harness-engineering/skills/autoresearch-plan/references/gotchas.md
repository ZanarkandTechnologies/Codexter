# Autoresearch Plan Gotchas

- Do not use this skill to execute experiments.
- Do not accept subjective metrics as the primary keep/discard signal.
- Do not skip Verify dry-run.
- Do not overwrite an existing session without reading it first.
- Do not let a broad editable scope include unrelated user work.
- Do not create a guard that optimizes the same number as Verify; guard is
  backpressure, not the goal.
