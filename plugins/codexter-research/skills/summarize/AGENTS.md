# `skills/summarize/AGENTS.md`

Rules for the `summarize` skill surface.

## Purpose

`skills/summarize/` owns fast URL/file/video summary guidance for the external
`summarize` CLI.

## Keep

- one-content-item summary workflows simple
- CLI usage concrete
- transcript-vs-summary distinction explicit
- fallback services documented, not assumed

## Do Not

- turn this into a broad research workflow
- duplicate content the user already pasted directly in chat
- pretend transcript extraction is guaranteed perfect
